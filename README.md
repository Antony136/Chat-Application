# Chat-Application

A real-time chat application built from scratch, inspired by WhatsApp. This project aims to provide a seamless and secure messaging experience.

- **Real-Time Messaging:** Instant one-on-one and group messaging.

- **Media Sharing:** Share images, videos, and other files.
- **Online Status:** See when users are online.
- **Voice and Video Calls:** High-quality calls using WebRTC.
